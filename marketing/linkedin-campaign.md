### Post 4: Quick Fixes (Helpful)
```
You can fix 50% of your website's accessibility issues today.

No developers needed. No budget required.

Here are 7 quick wins:

1. ADD ALT TEXT TO IMAGES
- Go through your product images
- Add descriptions: "Blue cotton t-shirt, front view"
- Takes 2 hours for most sites

2. CHECK YOUR COLOR CONTRAST
- Use WebAIM's free contrast checker
- Ensure 4.5:1 ratio for text
- Fix your most-used colors first

3. ADD SKIP LINKS
- Let keyboard users skip to main content
- One line of HTML + CSS
- 15-minute fix

4. LABEL YOUR FORMS
- Every input needs a visible label
- "Email" should be a label, not placeholder text
- Critical for checkout flows

5. TEST WITH KEYBOARD ONLY
- Unplug your mouse
- Can you complete a purchase?
- Note where you get stuck

6. ADD FOCUS INDICATORS
- Visible outline when tabbing through elements
- CSS one-liner: :focus { outline: 2px solid #000; }
- Often accidentally removed by designers

7. CAPTION YOUR VIDEOS
- YouTube auto-captions as a start
- Review and correct errors
- Huge impact for deaf/hard-of-hearing users

These 7 items address the most common EAA violations.

Which one are you tackling first?

#accessibility #webdev #quickwins #ecommerce #EAA
```
