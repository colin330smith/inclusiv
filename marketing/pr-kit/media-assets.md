## Asset Storage and Distribution

### Storage Location
- [ ] Set up dedicated folder (e.g., /press on website)
- [ ] Cloud backup (Google Drive, Dropbox)
- [ ] Version control for updates

### Distribution Method
- [ ] Public press page on website
- [ ] Private link for high-res assets
- [ ] Email delivery for custom requests

### File Naming Convention
```
inclusiv-[asset-type]-[variant]-[size].[format]

Examples:
inclusiv-logo-primary-1000px.png
inclusiv-headshot-colin-square.jpg
inclusiv-screenshot-dashboard-light.png
```
