# Inclusiv Support Operations - Launch Checklist

## Pre-Launch Setup

### Email Configuration

- [ ] Configure SPF/DKIM/DMARC records
- [ ] Test email deliverability

  - [ ] enterprise@tryinclusiv.com
  - [ ] partners@tryinclusiv.com

### Helpdesk Setup (Choose One)
- [ ] **Option A: Intercom** (Recommended for startups)
  - [ ] Create Intercom account
  - [ ] Install website widget
  - [ ] Set up Inbox workflows
  - [ ] Import email templates
  - [ ] Configure SLA automation
  
- [ ] **Option B: Freshdesk** (Budget-friendly)
  - [ ] Create Freshdesk account
  - [ ] Configure email channel
  - [ ] Set up automations
  - [ ] Import canned responses
  
- [ ] **Option C: Manual** (MVP approach)
  - [ ] Use shared Gmail/email inbox
  - [ ] Create labels for categorization
  - [ ] Use saved responses
  - [ ] Track in spreadsheet
